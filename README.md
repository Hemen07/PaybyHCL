# PaybyHCL

A simple app that demonstrates basic use cases as per HCL Hackathon.
+ Login Page
+ Account Summary
+ Account details
+ View All Beneficiaries
+ Funds Transfer



Technologies Used -
  
- MVVM with Clean Architecture
- Jetpack compose
- Kotlin
- Coroutines
- Hilt
- MockOtp Postman server
